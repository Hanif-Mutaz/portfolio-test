01 — about me

Bukan sekadar IT.
Saya membangun sistem.

Perjalanan saya tidak dimulai dari dunia programming. Saya mulai sebagai guru, lalu EDC field technician, lalu IT operator — setiap peran mengajarkan sesuatu yang berbeda tentang bagaimana sistem, manusia, dan proses saling berkaitan.

PM Monitoring System

Excel VBA · PostgreSQL · ConMas · Multi-view Dashboard
↓ 80%
waktu monitoring
Mesin-mesin di lantai produksi tidak punya visibilitas terpusat untuk Preventive Maintenance. Teknisi harus berjalan ke setiap mesin satu per satu untuk mengecek status — proses manual yang sering menyebabkan jadwal terlewat dan berdampak langsung pada kelangsungan produksi.
Saya memetakan kebutuhan tiap level stakeholder, membersihkan data counter mesin yang tidak konsisten, dan membangun Excel-VBA macro yang menghitung jadwal PM otomatis dari data counter ConMas. Hasilnya: satu dashboard terpusat yang menampilkan status OK / Warning / Danger / Overdue untuk semua mesin sekaligus.
real impact
Waktu monitoring: 1–2 jam → beberapa menit
Risiko keterlambatan PM berkurang ±70–80%
Puluhan mesin terpantau dalam satu layar
Management view untuk keputusan eksekutif
full-stack live

Dashboard Produksi Real-Time

Express.js · React · PostgreSQL · ConMas DB · Multi-shift
↓ 60–80%
waktu rekap
Data produksi tersimpan di ConMas tapi tidak punya tampilan visual apapun. Untuk tahu output vs target, harus buka data mentah secara manual — lambat, bergantung satu orang, dan selalu terlambat untuk koreksi.
Semua level organisasi kini melihat data produksi yang sama secara real-time sehingga deviasi target dapat diidentifikasi dan ditindaklanjuti jauh lebih cepat — tanpa menunggu rekap manual dari satu orang.
Dibangun menggunakan Express.js, React, dan PostgreSQL — backend menangani multi-view query lintas 3 shift dengan logic kolom dinamis (Jumat vs weekday), WIB timezone handling, dan 4 endpoint terpisah.
technical highlights
backend
UNION ALL multi-shift queries, dynamic column mapping (Jumat vs weekday), 4 endpoints
frontend
Arsitektur komponen React, custom hooks, tampilan operator / supervisor / manajemen
impact
Deviasi terdeteksi lebih awal, semua level baca data live yang sama
